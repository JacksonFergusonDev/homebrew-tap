# Jackson Ferguson's Homebrew Tap

This repository contains custom Homebrew formulae for my tools.

## Available Formulae

| Formula | Description |
| :--- | :--- |
| **git-pulsar** | Automated, paranoid git backups for students and casual coding. |

## Installation

To add this tap to your Homebrew installation:

```bash
brew tap JacksonFergusonDev/tap
```

To install a specific tool:

```bash
brew install git-pulsar
```
